# تريند سورية - وثيقة متطلبات المنتج

## المشكلة الأصلية
بناء تطبيق تجارة إلكترونية متعدد البائعين باسم "تريند سورية" مع:
- سوق إلكتروني لجميع أنواع المنتجات
- توصيل مجاني داخل سوريا
- نظام متعدد البائعين مع وثائق رسمية
- تتبع الطلبات
- تقييمات ومراجعات المنتجات
- نظام دردشة للمستخدمين/البائعين
- دفع عبر شام كاش

## الجمهور المستهدف
- المشترون في سوريا
- البائعون الذين يريدون بيع منتجاتهم

## التقنيات المستخدمة
- **Backend**: FastAPI, Python
- **Frontend**: React.js, TailwindCSS
- **Database**: MongoDB
- **Authentication**: JWT (Phone + Password)
