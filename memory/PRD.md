# تريند سوريا - وثيقة المتطلبات

## نظرة عامة
منصة تسوق إلكتروني متعددة البائعين تعمل في سوريا

## المواصفات الأساسية
- **الاسم**: تريند سوريا
- **الألوان**: برتقالي (#FF6B00) + أسود (#050505)
- **اللغة**: عربية (RTL)
- **التوصيل**: مجاني لجميع المحافظات
- **الدفع**: شام كاش (محاكاة)

## أنواع المستخدمين
1. **مشتري** - تسوق وشراء المنتجات
2. **بائع** - إدارة المنتجات والطلبات (يتطلب أوراق ثبوتية)
3. **مدير** - إدارة النظام والموافقة على البائعين
