## ملخص المشروع
تطبيق تجارة إلكترونية متكامل (Marketplace) باسم "تريند سورية" يدعم عدة أنواع من المستخدمين (بائعين، مشترين، موظفي توصيل، مدراء) مع نظام موافقات وعمولات ودفع إلكتروني.

## سلسلة الدفع 💰

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   العميل    │ ──▶ │  شام كاش   │ ──▶ │  تريند     │ ──▶ │ البائع +   │
│  (المشتري)  │     │  (تجريبي)  │     │  سورية     │     │ موظف توصيل │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘

مثال على طلب بقيمة 100,000 ل.س:
├── حصة البائع: 83,000 ل.س (تضاف للمحفظة)
└── عمولة تريند سورية (17%): 17,000 ل.س
    ├── أجرة التوصيل: 5,000 ل.س (تضاف لمحفظة موظف التوصيل)
    └── صافي ربح المنصة: 12,000 ل.س
```
